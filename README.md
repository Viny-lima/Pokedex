<h1 align="center" >Pokédex</h1>

<h2 align="center" >Objetivo</h2>

<div align="center" >  
  <p>Esse projeto tem por objetivo a criação de uma aplicação visual para fazer a listagem de Pokémons, com a exibição de suas características. Os dados serão coletados através   da API pública PokéAPI(https://pokeapi.co/)</p> <br>
